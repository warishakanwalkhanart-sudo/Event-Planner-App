✨ Features


Splash Screen — Beautiful landing screen with an inspirational quote and Explore button
Upcoming Events — View all upcoming events in a visually rich card layout with event images, dates, and locations
Past Events — Browse previously held events with full details and reference photos
Create Event — Add new events with title, location, category (Birthday, Dinner, etc.), and color theme
Event Planning Guide — In-app article covering complete event planning steps including basic planning, theme selection, decor, seating arrangements, creative additions, and checklists
Local Data Storage — Events are saved locally for persistent access across sessions
Responsive UI — Clean, colorful, and intuitive interface that works on both Android and iOS

🚀 Getting Started

Prerequisites


Node.js installed
Expo Go app installed on your phone (Android / iOS)


Installation

1. Clone the repository

bashgit clone https://github.com/warishakanwalkhanart-sudo/event-planner.git
cd event-planner

2. Install dependencies

bashnpm install

3. Start the development server

bashnpx expo start

4. Scan the QR code with Expo Go app on your phone — app will load instantly!
